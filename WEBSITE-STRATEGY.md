# Asia Weighing 独立站分析与规划

## 1. 商业定位

- 公司：Changzhou Asia Weighing Apparatus Co., Ltd.
- 品牌：Asia Weighing
- 定位：工业称重设备源头制造商与 OEM/ODM 解决方案提供商
- 主要客户：进口商、经销商、工程商和 EPC 公司
- 核心市场：中东、非洲、南美，以及除印度外的其他英语采购市场
- 首要目标：获取有效询盘，其次是 SEO 获客和 Google Ads 落地页转化
- 首期产品：Bench Scale、Floor Scale、Custom Industrial Weighing Systems

网站不应定位成低价产品目录。核心购买理由应当是：能够根据应用、量程、精度、尺寸、材质和接口要求完成产品配置，并提供稳定的项目沟通。

## 2. 首页转化逻辑

1. 首屏说明产品范围、目标客户与 OEM/ODM 能力。
2. 三个产品入口帮助采购商快速判断是否匹配。
3. OEM/ODM 流程降低定制采购的不确定性。
4. 应用场景帮助非技术采购商表达需求。
5. 工厂事实建立制造能力信任，但不虚构认证、客户或项目。
6. 采购指南教育客户提供有效参数。
7. FAQ 处理常见疑虑。
8. RFQ 表单收集国家、产品、用途、邮箱和 WhatsApp。

## 3. 建议网站结构

| 页面 | 建议 URL | 主要作用 |
| --- | --- | --- |
| 首页 | `/` | 品牌定位与询盘转化 |
| 产品中心 | `/products` | 产品分类入口 |
| 台秤 | `/products/bench-scales` | 承接 bench scale 商业词 |
| 地磅/平台秤 | `/products/floor-scales` | 承接 floor/platform scale 商业词 |

| 工厂 | `/factory` | 车间、质检、包装和交付证据 |
| 博客 | `/blog` | 选型、采购和维护内容 |
| 联系/询价 | `/request-a-quote` | 完整 RFQ |

## 4. SEO 与广告分工

- 新站优先承接解决方案、采购指南、广告落地页与 RFQ。
- 旧站继续运营时，不复制产品描述、标题、文章或案例。
- `manufacturer`、`supplier`、`factory`、`price` 等同类词合并到一个高质量产品中心页。
- `truck scale` 暂不作为首页主推产品；产品资料、参数和交付能力确认后再建立商业页。
- 首期 Google Ads 建议拆分为 Bench Scale、Floor Scale、OEM/ODM 三组，并分别进入对应落地页。
- 国家页只有在具备当地交付、认证或真实项目信息时才发布。

## 5. 内容证据规则

- 可以使用：2006 年成立、30,000+ m²、30+ 团队、真实画册产品和已确认联系方式。
- 认证信息、出口数量、专利、交期、质保和客户案例必须在内部资料确认后再发布。
- 每个产品详情页应包含量程、精度、尺寸、材质、传感器、仪表、接口、定制范围、包装及询价入口。
- 技术文章注明审核人和更新时间，方便搜索引擎及 AI 搜索引用。
