# Asia Weighing Next.js Homepage

Responsive B2B lead-generation homepage based on the supplied company brief,
catalog, logo and SEO keyword plan.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Before production

- Replace the temporary `asiaweighing.com` metadata domain with the purchased domain.
- Connect the RFQ form to the chosen email/CRM service.
- Verify all company figures and add only approved certification files.
- Add privacy policy, terms and consent text for the target markets.
